The What Behind Machine Learning

Machine Learning is the process of making machines act without explicitly being programmed to accomplish a task.
Let us now talk about the three most common types of Machine Learning Models.

Regression

The goal of Regression analysis is to predict a continuous outcome variable.
Become familiar with the Linear Regression Models, Bayesian Models, K-Nearest Neighbors Models(used for both regression and classification).

Classification

The goal of classification is to predict the class or category that the data will fall into.
Some popular classification models are Logistic Regression, Naive Bayes, Stochastic Gradient Descent, K-Nearest Neighbors, Decision Trees, Random Forest, Support Vector Machine.

Reinforcement

The goal of reinforcement learning is that the agent(learning system) can observe the environment and select/perform actions that get rewards in return.
